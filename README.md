# ✨ Key Features

🚀 **Automatic Face Detection**  
Detects human faces using OpenCV’s Haar Cascade model.

🎭 **Face Anonymization**  
Applies Gaussian blur to obscure facial identities.

🎚 **Adjustable Blur Strength**  
Control blur intensity for stronger anonymization.

🔍 **Detection Sensitivity Controls**  
Tune parameters for better detection accuracy.

🎥 **Video Upload Support**  
Supports **MP4, MOV, and AVI** formats.

📊 **Real-time Processing Feedback**  
Frame-by-frame processing with a progress bar.

🖥 **Before & After Preview**  
Compare the original and anonymized video directly in the UI.

⬇️ **Download Processed Video**  
Export the sanitized video in one click.

🔒 **100% Local Processing**  
No cloud upload — all computation happens on your machine.

🎨 **Modern UI Design**  
Dark theme with **glassmorphism UI** and an interactive sidebar.
